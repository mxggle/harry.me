日语的量词（助数詞 - じょすうし）是用来数不同种类物品或事件的词语。 **为什么日语量词会有音变？** 日语量词的音变（「音便」）主要源于发音的流畅性和历史演变。当数字与特定量词组合时，为了**发音更顺口**，部分音节会发生浊化（如「ひき→びき」）或促音变（如「はち＋かい→はっかい」）。这些变化并非随意，而是有规律可循的！

**📌 音变三大原因：**

1. **发音省力**：如「いち＋かい→いっかい」，省略「ち」的发音更流畅。
2. **历史习惯**：部分音变保留自古代日语发音（如「はつ→はっ」）。
3. **区分意义**：避免歧义（如「しち」和「はち」需音变区分）。

注： 本手册分两个模块

1. 暴力穷举：列出所有量词以及音变，方便查阅
2. 规则总结：按照所有的量词音变情况，总结了一些规则，方便记忆

会发生音变的数字我会用红色标注

# 暴力穷举

## 发生音变的量词

### 1.🍊 ～つ - 个（通用）

这是日语固有的数数方法，只能数到10。当不确定用哪个量词，或者数抽象事物时常用。

- **1: ひとつ - 一个**
- **2: ふたつ - 两个**
- **3: みっつ - 三个**
- **4: よっつ - 四个**
- **5: いつつ - 五个**
- **6: むっつ - 六个**
- **7: ななつ - 七个**
- **8: やっつ - 八个**
- **9: ここのつ - 九个**
- **10: とお - 十个**
- いくつ - 几个？

**使用规律:**

1. 想象一下，其实这个读起来最有节奏感的（动つ打つ🤘）记住不难
2. 日本传统数法，主要用于10个以内的小件物品或抽象事物（例：问题有一个 - 問題がひとつある）。
3. 11个以上通常用“～個 (こ)”。

### 2\. 📦 ～個 (こ) - 个（小物体）

用于数小而圆的物体，或一般物体，非常通用。

- **1: いっこ - 一个**
- 2: にこ - 两个
- 3: さんこ - 三个
- 4: よんこ - 四个
- 5: ごこ - 五个
- **6: ろっこ - 六个**
- 7: ななこ / しちこ - 七个
- **8: はっこ / はちこ - 八个**
- 9: きゅうこ - 九个
- **10: じゅっこ / じっこ - 十个**
- **100: ひゃっこ - 一百个**
- 何個 (なんこ) - 几个？

**使用规律:**

1. 用于苹果、石头、盒子等较小、形状固定的物体。
2. 当“～つ”不能用时（11个以上），这个量词很通用。
3. 数字1, 6, 8, 10, 100后常发生促音变。

### 3\. 🙆‍♂️ ～人 (にん / り) - 人

用于数人数。

- **1: ひとり - 一个人**
- **2: ふたり - 两个人**
- 3: さんにん - 三个人
- **4: よにん - 四个人 (注意！不是“しにん”，shinin是死人的意思)**
- 5: ごにん - 五个人
- 6: ろくにん - 六个人
- 7: ななにん / しちにん - 七个人
- 8: はちにん - 八个人
- **9: きゅうにん / くにん - 九个人**
- 10: じゅうにん - 十个人
- 何人 (なんにん) - 几个人？

**使用规律:**

1. 1人和2人有特殊读法“ひとり”、“ふたり”。
2. 4人读“よにん”以避开“しにん (死人)”的不吉利发音。
3. 7人、9人有多种读法，但“ななにん”、“きゅうにん”更常用。

### 4\. 🐶～匹 (ひき / ぴき / びき) - 只（小动物、鱼、虫）

用于数狗、猫、鱼、虫等较小的动物。

- **1: いっぴき - 一只**
- 2: にひき - 两只
- **3: さんびき - 三只**
- 4: よんひき - 四只
- 5: ごひき - 五只
- **6: ろっぴき - 六只**
- 7: ななひき / しちひき - 七只
- **8: はっぴき / はちひき - 八只**
- 9: きゅうひき - 九只
- **10: じゅっぴき / じっぴき - 十只**
- **何匹 (なんびき) - 几只？**

**使用规律:**

1. 根据数字末尾音节，量词读音在“ひき”、“ぴき”、“びき”间变化。
2. 数字1, 6, 8, 10后：“～ぴき” (促音+半浊音)。
3. 数字3, 及问“几只(何)”时：“～びき” (浊音)。
4. 其他数字后：“～ひき”。

### 6\. 🐦 ～羽 (わ / ば) - 只（鸟类、兔子）

### 数字 **4 (四)** 的音变和特殊读法总结

数字“四”通常有三种读法与量词结合：「よん」、「よ」、「し」。其中「し」因为与“死 (し)”同音，在很多情况下会被避开。

1. **读作 「よ」 的情况 (最特殊，需要记忆)：**
    - **～人 (にん):** 4人 → **よにん** (四个人)
        - _这是为了避免「しにん」(死人) 的发音。_
    - **～時 (じ) (时间点):** 4時 → **よじ** (四点钟)
        - **因此，～時間 (じかん) (时长):** 4時間 → **よじかん** (四小时)
    - **～年 (ねん):** 4年 → **よねん** (四年)
    - **～日 (か) (日期):** 4日 → **よっか** (四号 / 四天) - _这是和语固有读法，不仅仅是「よ」。_
    - **～つ (通用量词):** 4つ → **よっつ** (四个) - _和语固有读法。_
2. **通常读作 「よん」 的情况 (较普遍)：**
    - 当不属于上述特殊情况时，大部分量词前，“四”都读作「よん」。
    - 例如：
        - ～個 (こ): よんこ (四个)
        - ～匹 (ひき): よんひき (四只)
        - ～本 (ほん): よんほん (四根)
        - ～枚 (まい): よんまい (四张)
        - ～冊 (さつ): よんさつ (四册)
        - ～台 (だい): よんだい (四台)
        - ～階 (かい): よんかい (四层)
        - ～回 (かい): よんかい (四次)
        - ～杯 (はい): よんはい (四杯)
        - ～歳 (さい): よんさい (四岁)
        - ～足 (そく): よんそく (四双)
        - ～着 (ちゃく): よんちゃく (四套)
        - ～分 (ぷん): よんぷん (四分钟)
3. **读作 「し」 的情况 ：**
    - 月份：四月 (しがつ) **记忆重点：** 记住「よにん」、「よじ(かん)」、「よねん」、「よっか」、「よっつ」这几个特殊的「よ」或和语读法，其余多数为「よん」。

### 数字 **9 (九)** 的音变和特殊读法总结

数字“九”通常有「きゅう」和「く」两种读法。「く」因为与“苦 (く)”同音，在某些情况下会被「きゅう」取代，但也有「く」是固定读法的情况。

1. **读作 「く」 的情况 (特定，需要记忆)：**
    - **～時 (じ) (时间点):** 9時 → **くじ** (九点钟)
        - **因此，～時間 (じかん) (时长):** 9時間 → **くじかん** (九小时)
    - **～日 (か) (日期):** 9日 → **ここのか** (九号 / 九天) - _和语固有读法，不仅仅是「く」。_
    - **～つ (通用量词):** 9つ → **ここのつ** (九个) - _和语固有读法。_
2. **通常读作 「きゅう」 的情况 (较普遍，优先使用)：**
    - 当不属于上述特殊情况时，大部分量词前，“九”都读作「きゅう」。这是现代日语中更常用的读法，以避免“苦”的联想。
    - 例如：
        - ～個 (こ): きゅうこ (九个)
        - ～人 (にん): きゅうにん (九个人) (文档中也提到了 くにん，但 きゅうにん 更常用)
        - ～匹 (ひき): きゅうひき (九只)
        - ～頭 (とう): きゅうとう (九头)
        - ～羽 (わ): きゅうわ (九只)
        - ～本 (ほん): きゅうほん (九根)
        - ～枚 (まい): きゅうまい (九张)
        - ～冊 (さつ): きゅうさつ (九册)
        - ～台 (だい): きゅうだい (九台)
        - ～階 (かい): きゅうかい (九层)
        - ～回 (かい): きゅうかい (九次)
        - ～杯 (はい): きゅうはい (九杯)
        - ～歳 (さい): きゅうさい (九岁)
        - ～足 (そく): きゅうそく (九双)
        - ～着 (ちゃく): きゅうちゃく (九套)
        - ～分 (ふん): きゅうふん (九分钟)
        - ～週間 (しゅうかん): きゅうしゅうかん (九周)
        - ～ヶ月 (かげつ): きゅうかげつ (九个月)
        - ～年 (ねん): きゅうねん (九年) (文档中也提到了 くねん，但 きゅうねん 更常用)
3. **读作 「くにん」、「くねん」的情况：**
    - ～人 和 ～年 时，九也有「くにん」和「くねん」的读法。「きゅうにん」和「きゅうねん」更常用。

**记忆重点：** 记住「くじ(かん)」、「ここのか」、「ここのつ」这几个特殊的「く」或和语读法，其余多数情况下优先使用「きゅう」。
